engulf

Skills:Game Design, Interaction Design, Python
Roles:Design & Implementation
Status:Complete

Endless gesture-driven consumption

Engulf is a frantic quest to consume nutrients and grow larger while escaping a cloud of enemies. The gameplay and control scheme are finely tuned to take advantage of the Kinect; you must change your avatar's appearance and size in order to eat food of the appropriate color while dodging between narrow gaps formed by enemies harrowing you at every turn.

This day-long project began as an attempt to make a demo for my Python drawing library, pyshiva. As soon as I connected the library to the Kinect, I knew that I wanted to make a game.

pyShiva

Collaborators:Berit Johnson, Diana Vermilya, Charles Gwennap
Skills:API Design, C, Cython, Python
Roles:Design & Implementation
Status:Under Active Development

Simple 2D vector graphics for Python

With pyshiva, graphics programming is fun again. The pyshiva Python module provides a simple, object-oriented programming interface for creating stunning 2D graphics for realtime data visualization, games, and general 2D experimentation.

Technical details: pyshiva is a high level Cython extension built on the OpenGL-based ANSI-C ShivaVG graphics library and the cross-platform GLFW windowing library. It abstracts the process of drawing objects via an object-oriented API that enables programmers to create persistent vector graphics objects.
